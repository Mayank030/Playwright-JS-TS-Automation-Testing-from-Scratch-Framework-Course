# Playwright Automation Testing Course

> **Course Updated:** June 2026
> Includes **Playwright AI Agents**, **MCP (Model Context Protocol)**, and practical demonstrations.

## 📖 About the Course

This comprehensive Playwright course is designed to take you from beginner to advanced level, covering everything from JavaScript fundamentals to building enterprise-grade automation frameworks.

No prior programming experience is required. A dedicated JavaScript section is included to help beginners build a strong foundation before diving into Playwright automation.

The course follows a **hands-on learning approach**, using a real-world e-commerce application to automate practical testing scenarios commonly found in modern React and Angular applications.

---

# 🚀 What You'll Learn

* JavaScript and TypeScript fundamentals required for Playwright
* Web UI automation using Playwright
* Playwright locators and automation methods
* End-to-end functional testing
* API testing using Playwright
* Combining UI and API testing in a single automation flow
* Creating and deleting test data using APIs
* Network request and response interception
* Network mocking
* Session management using cookies and API responses
* Bypassing UI login for faster execution
* Playwright Inspector
* Trace Viewer
* Code Generator (Record & Playback)
* AI Agents with Playwright
* MCP (Model Context Protocol) integration
* GitHub Copilot integration in VS Code
* Visual testing
* Mobile device emulation
* Screenshots and video recording
* Debugging and logging
* CI/CD integration with Jenkins
* Docker-based test execution
