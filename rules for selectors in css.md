**Rules to identify elements on a page  using CSS selectors**

1\) If ID is present:
css -> tagname#id or #id

2\) if class attribute is present:
css -> tagname.class or .class

3\) Write css based on any Attribute: 
css -> \[attribute='value']
or
\tagname[attribute='value']

4\) Write css with traversing from Parent to Child:
css -> parenttagename [space] childtagname

5\) if needs to write the locator based on the text or psudo classes 
"text=''"
or
"tag:has-text("text")"  --> example "h3:has-text('COAT')"
